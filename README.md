# Speech To Text
A Speech To Text website powered by Tensorflow.js!

## Usage
- Execute `npm install`
- To start `npm run start`

## Working words
- Yes
- No
- Hello
- Hey
- Cool
- LOL
- Hoi
- Help

## Requesting new words
Create a **issue** to submit a word to be added!

## Info
**Model size:** ~5 MB
